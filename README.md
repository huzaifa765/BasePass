## What is BasePass

BasePass is a permanent onchain identity protocol on Base mainnet. Create your Web3 profile, get endorsed by community, earn achievements, and build your onchain reputation.

## Identity Features

- Permanent onchain profile
- Skill endorsements from community
- Achievement badges
- Tier system — Bronze to Diamond
- Follower and following system

## Why BasePass

- No server — fully onchain
- Censorship resistant
- Permanent — your identity forever
- Community driven endorsements
